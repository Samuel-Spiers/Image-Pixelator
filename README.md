# Image-Pixelator
A simple image processing program that uses a user-defined scaler value to pixelate an image and allow the user to save the new version.
"Pixelate Factor" determines the amount to pixelate the image. Higher means more pixelation
"Minimum Dimension" determines the minimum width/height of the new image (The image comes out the same resolution as before, this refers to the visible "pixelated" pixels
